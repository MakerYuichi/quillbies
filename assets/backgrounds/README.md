# Background Images

## 📍 Place Your Images Here

### For Screen 1 (Welcome)
- **File name**: `welcome-bg.png`
- **Size**: 1080x1920 or higher (portrait orientation)
- **Format**: PNG or JPG

This is where your background image should go!

## Quick Start

1. Take your background image
2. Rename it to `welcome-bg.png`
3. Drop it in this folder
4. The Welcome screen will automatically use it

## Image Tips

- Use high-resolution images for crisp display on all devices
- Portrait orientation works best for mobile
- The app adds a 40% dark overlay for text readability
- Test with different images to find what works best
