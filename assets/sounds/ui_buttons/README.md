# UI Feedback Sounds

Button and interface interaction sounds.

## Required Files:
- `button_click.mp3` - Tapping buttons
- `toggle_on.mp3` - Switching features on
- `toggle_off.mp3` - Switching features off
- `menu_open.mp3` - Opening menus
- `menu_close.mp3` - Closing menus

## Specifications:
- Format: MP3
- Duration: 0.1-0.5 seconds
- Volume: Normalized to -16 LUFS
- Sample Rate: 44.1kHz
- Should be subtle and non-intrusive
