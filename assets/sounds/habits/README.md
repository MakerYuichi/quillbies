# Habit Logging Sounds

Sounds for logging daily habits.

## Required Files:
- `water_log.mp3` - Drinking water sound
- `meal_log.mp3` - Eating sound
- `exercise_log.mp3` - Exercise completion
- `sleep_log.mp3` - Sleep tracking
- `clean_room.mp3` - Room cleaning sound

## Specifications:
- Format: MP3
- Duration: 0.5-2 seconds
- Volume: Normalized to -14 LUFS
- Sample Rate: 44.1kHz
- Should be satisfying and rewarding
