# Character Sounds

Hamster personality sounds for different states.

## Required Files:
- `hamster_happy.mp3` - When habits logged
- `hamster_sad.mp3` - When missing goals
- `hamster_eating.mp3` - When logging meals
- `hamster_drinking.mp3` - When logging water
- `hamster_sleeping.mp3` - When sleeping

## Specifications:
- Format: MP3
- Duration: 0.5-2 seconds
- Volume: Normalized to -14 LUFS
- Sample Rate: 44.1kHz
- Should be cute and expressive
