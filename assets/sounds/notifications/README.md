# Notification Sounds (Priority 1)

Short alert sounds for important events.

## Required Files:
- `session_start.mp3` - "Focus session starting!"
- `session_end.mp3` - "Great job! Session complete"
- `break_start.mp3` - "Time for a break!"
- `break_end.mp3` - "Break's over, back to work"
- `warning_alert.mp3` - "You're getting distracted"
- `task_complete.mp3` - "Task finished!"
- `error.mp3` - "Something went wrong"

## Specifications:
- Format: MP3
- Duration: 1-3 seconds
- Volume: Normalized to -14 LUFS
- Sample Rate: 44.1kHz
