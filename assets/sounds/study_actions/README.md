# Study Action Sounds

Study-specific interaction sounds.

## Required Files:
- `page_turn.mp3` - Flipping pages
- `pen_click.mp3` - Writing/checking tasks
- `keyboard_type.mp3` - Typing sounds
- `timer_tick.mp3` - Soft timer ticks (optional)
- `goal_reached.mp3` - When study goal achieved

## Specifications:
- Format: MP3
- Duration: 0.2-1 seconds
- Volume: Normalized to -16 LUFS
- Sample Rate: 44.1kHz
- Should be satisfying and motivating
