# Background Music (Priority 2)

Looping study music for focus sessions.

## Required Files:
- `lofi_study_1.mp3` - Chill lo-fi beats
- `lofi_study_2.mp3` - Alternative lo-fi track
- `classical_piano.mp3` - Soft classical piano
- `ambient_focus.mp3` - Atmospheric ambient
- `nature_sounds.mp3` - Rain/forest sounds

## Specifications:
- Format: MP3
- Duration: 2-5 minutes (seamless loop)
- Volume: Normalized to -18 LUFS (quieter than notifications)
- Sample Rate: 44.1kHz
- Should loop seamlessly
