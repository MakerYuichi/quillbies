# Achievement Sounds

Reward and unlock sounds for milestones.

## Required Files:
- `coin_earned.mp3` - Earning Q-coins
- `level_up.mp3` - Level progression
- `streak_milestone.mp3` - 7/30/100 day streaks
- `achievement_unlock.mp3` - Badge unlocked
- `premium_unlock.mp3` - Premium feature unlocked

## Specifications:
- Format: MP3
- Duration: 1-3 seconds
- Volume: Normalized to -12 LUFS (slightly louder for celebration)
- Sample Rate: 44.1kHz
- Should be celebratory and rewarding
