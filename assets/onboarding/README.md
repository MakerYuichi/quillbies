# Onboarding Assets

## 📍 Place Your Images Here

### Welcome Screen Hamster
- **File name**: `welcome-hamster.png`
- **Size**: 220x220px or higher (square)
- **Format**: PNG with transparent background
- **Style**: Cute, friendly hamster illustration

This hamster appears on the welcome screen between the text and buttons!

## Image Specifications

### welcome-hamster.png
- **Recommended size**: 400x400px (will be displayed at 220x220)
- **Background**: Transparent PNG
- **Style**: Cute, welcoming, friendly
- **Pose**: Standing, waving, or looking friendly
- **Colors**: Should work well against dark overlay

## Tips

- Use high-resolution images (2x or 3x) for crisp display
- Transparent backgrounds work best
- Test on both light and dark backgrounds
- Keep file sizes reasonable (<500KB per image)

## Quick Start

1. Create or find a cute hamster illustration
2. Export as PNG with transparent background
3. Name it `welcome-hamster.png`
4. Drop it in this folder
5. The welcome screen will automatically display it!
