# Deadline Management System

## Overview
The deadline management system helps users track and work towards their academic and personal deadlines through focused study sessions.

## Features

### 1. Deadline Creation
- **Task Name**: What needs to be completed
- **Due Date**: Interactive calendar picker (required)
- **Due Time**: Interactive time picker (optional)
- **Priority**: High (🔴), Medium (🟡), Low (🟢)
- **Estimated Hours**: How much work is needed
- **Category**: Study, Work, Project, Other

### 2. Deadline Organization
- **Urgent**: Due within 3 days (red section)
- **Upcoming**: Due in more than 3 days (yellow section)
- **Completed**: Finished tasks (green section)

### 3. Focus Session Integration
- Start focus sessions specifically for a deadline
- Automatically track work hours towards deadlines
- Real-time progress updates during sessions
- Work completion tracking (hours completed / total hours)

### 4. Progress Tracking
- Visual progress bars showing completion percentage
- Automatic completion when estimated hours are reached
- Work history and time logging
- Integration with existing study habit system

## How It Works

### Creating a Deadline
1. Tap "➕ CREATE NEW DEADLINE" on the Focus screen
2. Fill in the task name and estimated hours
3. Use the calendar picker to select due date
4. Optionally use the time picker for specific time
5. Select priority level and category
6. Tap "✓ Create" to save

### Working on Deadlines
1. Find your deadline in the Urgent or Upcoming sections
2. Tap "🎯 Focus on This" to start a focused session
3. Complete your Pomodoro session as normal
4. Work hours are automatically added to the deadline

### Tracking Progress
- Each deadline shows: `X.Xh / Yh completed`
- Progress is updated in real-time during focus sessions
- Deadlines auto-complete when all estimated work is done

## Data Persistence
- All deadlines are saved automatically
- Progress persists between app sessions
- Work history is maintained
- Integration with existing user data system
