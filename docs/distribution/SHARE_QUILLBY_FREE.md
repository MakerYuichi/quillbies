# 🎉 Share Quillby for FREE - No Apple Developer Needed!

## 🚀 Fastest FREE Method (2 Commands)

```bash
cd quillby-app
npx expo publish
```

**That's it!** Share the QR code with anyone. Works on both Android & iPhone! 🎊

---

## 📱 How Users Install (2 Steps)

### Step 1: Install Expo Go (Free App)
- **Android:** [Play Store - Expo Go](https://play.google.com/store/apps/details?id=host.exp.exponent)
- **iPhone:** [App Store - Expo Go](https://apps.apple.com/app/expo-go/id982107779)

### Step 2: Scan Your QR Code
- Open Expo Go
- Tap "Scan QR Code"
- Scan your code
- Quillby opens! 🐹
